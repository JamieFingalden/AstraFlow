# AstraFlow（星流）- AI智能简历与求职分析平台

## 项目简介

AstraFlow是一款以"AI智能简历与求职分析"为核心的创新型Web平台，目标是帮助用户快速生成高质量的简历、分析岗位匹配度，并提供智能求职辅助建议。

### 核心功能

- 📄 **智能简历生成** - 基于AI技术，自动生成专业简历，提供优化建议
- 📈 **岗位匹配分析** - 输入岗位JD，智能分析匹配度，提供针对性建议
- 📊 **职业画像展示** - 可视化展示职业能力，帮助规划职业发展路径
- 🎯 **AI优化建议** - 连接心流平台API，提供专业的求职建议

## 技术栈

### 前端
- **框架**: Vue 3 + Vite
- **路由**: Vue Router 4
- **HTTP客户端**: Axios
- **图表**: Chart.js
- **构建工具**: Vite

### 后端（计划）
- **框架**: Gin
- **AI服务**: 心流平台API

## 项目特色

### 🎨 视觉设计
- **星河主题**: 采用星系蓝、流光、霓虹线条等科幻UI元素
- **响应式设计**: 完美适配桌面端和移动端
- **流畅动画**: 卡片层次动效、页面过渡动画、悬浮效果

### ⚡ 性能优化
- **懒加载**: 组件和图片懒加载优化
- **代码分割**: 路由级别的代码分割
- **缓存策略**: API响应缓存和静态资源优化

## 快速开始

### 环境要求
- Node.js 16+
- npm 或 yarn

### 安装依赖
```bash
npm install
```

### 开发环境
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 核心页面

### 1. 首页 (`/`)
- 品牌展示和英雄区动画
- 特色功能介绍
- 响应式导航菜单

### 2. 简历生成 (`/resume`)
- 用户信息输入表单
- 实时简历预览
- AI优化建议展示

### 3. 岗位分析 (`/job-analysis`)
- 岗位JD输入
- 匹配度可视化分析
- 优化建议和推荐

### 4. 职业画像 (`/profile`)
- 技能雷达图

- 数据可视化展示
